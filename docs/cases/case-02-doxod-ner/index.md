# Кейс 02: NER-маскирование ПДн в документах (ИК «ДОХОДЪ»)

## Краткое описание

**Домен:** финансовые услуги — инвестиционная компания «ДОХОДЪ» (брокер / управляющая компания).

**Задача:** автоматически находить и маскировать персональные данные (ФИО, паспорта, ИНН, СНИЛС, номера счетов, суммы, контакты) в документах — договорах, отчётах, приложениях — чтобы безопасно передавать их внешним подрядчикам в рамках интеграций.

**Итог одной строкой:** дообученный **GliNER** + собственный PDF-редакт-пайплайн на opensource-стеке (pdfplumber + reportlab + pypdf) на on-prem без интернета; **Recall 0.90–0.95**, **F1 ≈ 0.90**, сотни документов в день, ручной процесс маскирования заменён автоматическим.
