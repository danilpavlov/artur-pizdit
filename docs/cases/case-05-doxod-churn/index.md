# Кейс 05 (мини): Прогноз оттока клиентов (Churn) — ИК «ДОХОДЪ»

!!! info "Мини-кейс"
    Классический табличный ML-кейс. Компактная версия: 3 страницы вместо 7 — обзор, флоу, детали.

## Краткое описание

**Домен:** розничные инвестиции, retention-команда ИК «ДОХОДЪ».

**Задача:** за 30–90 дней до фактического оттока предсказывать, кто из клиентов с высокой вероятностью закроет счёт или перестанет торговать, чтобы retention-команда работала с «под риск» заранее, а не постфактум.

**Итог одной строкой:** бинарный классификатор **CatBoost/LightGBM** по транзакционным, портфельным и поведенческим фичам; ночной скоринг в Airflow; результат уходит в CRM-витрину ретеншн-команде.

## Формулировка для резюме

> **Разработал** модель прогноза оттока клиентов инвесткомпании на **CatBoost/LightGBM** (транзакции, портфель, поведение в ЛК, обращения в поддержку). Использовал **time-aware split**, **PR-AUC** как главную метрику из-за дисбаланса, интерпретацию через **SHAP**. Скоринг запускается ночью в **Airflow**, результат уходит retention-команде в CRM. **Ранний список «под риск» за 30–90 дней до фактического оттока**.
