# Кейс 04: Сегментация клиентов и Next Best Offer (ИК «ДОХОДЪ»)

## Краткое описание

**Домен:** розничные инвестиции, CRM-маркетинг инвестиционной компании «ДОХОДЪ».

**Задача:** перестать слать маркетинговые коммуникации «веером» по всей клиентской базе и начать подбирать каждому клиенту **Next Best Offer** — инвестпродукт (акции, облигации, ПИФ/БПИФ, ИИС, ДУ), на который он с наибольшей вероятностью откликнется.

**Итог одной строкой:** гибрид **HDBSCAN-сегментации** по поведенческим + портфельным + KYC-фичам и **CatBoost/LightGBM-скоринга** вероятности отклика на каждый продукт; работает в двух режимах — **ночной batch** в Airflow + **realtime-досчёт** при ключевых событиях клиента; интерпретация для маркетинга через SHAP; бизнес-эффект измерен в A/B-тесте.
