# Команда и роли

**Состав:** полноценная кросс-функциональная команда, 5+ человек.

## Состав

| Роль | Зона ответственности |
|---|---|
| **Data Scientist (Артур)** | Feature engineering, HDBSCAN-сегментация, обучение и валидация CatBoost/LightGBM-скоров на каждый продукт, метрики (ROC-AUC/PR-AUC, silhouette), интерпретация моделей через SHAP, портреты сегментов |
| **Data Engineer** | Витрины данных для фичей и результатов скоринга, Airflow-DAG'и batch-пайплайна, feature store, качество источников |
| **ML Engineer** | Realtime-досчёт при ключевых событиях клиента, деплой моделей, мониторинг drift'а и качества, интеграция с триггерной шиной (Kafka/webhook) |
| **Маркетолог / CRM-менеджер** | Заказчик; формулирует гипотезы по сегментам и офферам, принимает портреты сегментов, проводит маркетинговые кампании, даёт обратную связь |
| **Продуктовый аналитик** | A/B-тесты, конверсионная аналитика, uplift-анализ, валидация бизнес-эффекта |
| **Product Manager** | Приоритизация продуктов для NBA, согласование со стейкхолдерами, скоуп и сроки |

## Моя зона ответственности (Артур, DS)

- **Собрал обучающую выборку** из транзакций, портфеля, KYC и истории откликов; зафиксировал правила построения фичей «на дату T».
- **Спроектировал feature store** вместе с DE — чтобы фичи на обучение и инференс считались одинаково и «на момент».
- **Построил HDBSCAN-сегментацию:** подобрал параметры, оценил через silhouette / Davies-Bouldin, сделал **портреты сегментов** через описательную статистику + глобальные SHAP-важности.
- **Обучил и валидировал** отдельные CatBoost/LightGBM-модели на каждый продукт (акции/облигации, ПИФ/БПИФ, ИИС, ДУ); работал над дисбалансом классов через `scale_pos_weight` и PR-AUC как метрику.
- **Настроил time-aware split** и бенчмарк на target leakage (см. [ретро](retro.md)).
- **Реализовал NBA-шаг:** argmax по вектору скоров + согласовал с маркетингом и комплаенсом слой бизнес-правил (риск-профиль, KYC, стоп-листы).
- **Объяснил результаты** маркетингу через SHAP-портреты и демо на конкретных клиентах.
- **Совместно с аналитиком** готовил дизайн A/B-теста (контрольная группа, стратификация, сроки) и интерпретировал его результаты.

## Взаимодействие

- **С Data Engineer** — плотно на этапе построения feature store и Airflow-DAG'а.
- **С ML Engineer** — при деплое, настройке realtime-ветки, мониторинге.
- **С маркетологом / CRM** — регулярно: презентация сегментов, обсуждение гипотез, сбор фидбэка по результатам рассылок.
- **С аналитиком** — на A/B-тестах и интерпретации результатов.
- **С PM** — на приоритизации продуктов и скоупе моделей.
