# Достижения

## Бизнес-результаты

- **Маркетинг перешёл от веерных рассылок к персональным Next Best Offer.** Каждый клиент получает оффер, который система посчитала для него наиболее вероятным.
- **Рост конверсии в сегментах** по сравнению с контрольной группой в A/B-тесте (точные числа — TODO).
- **Снижение «шумовых» коммуникаций.** Клиенты, для которых ни один продукт не проходит по бизнес-правилам / имеет низкий скор, исключаются из рассылки — меньше отписок и жалоб.
- **Единая витрина `client → offer + explanation`** для всех каналов: email, push, CRM-обзвоны, личный кабинет.
- **Объяснимость для маркетинга.** Каждый оффер сопровождается коротким «портретом сегмента» + топ-факторами SHAP — маркетолог понимает, почему клиент попал под этот оффер.

## Технические результаты

- **Сегментация HDBSCAN:** оценка через **silhouette** и **Davies-Bouldin**; сегменты интерпретируются через описательные статистики и SHAP-важности.
- **Скоринг на продукты:** отдельные **CatBoost/LightGBM** на каждый продукт (акции/облигации, ПИФ/БПИФ, ИИС, ДУ).
- **Метрики скоринга:** **ROC-AUC** и **PR-AUC** на time-aware hold-out'е (последняя важнее из-за сильного дисбаланса классов).
- **Uplift** относительно контрольной группы в A/B-тесте — как главная бизнес-метрика, подтверждающая, что модель действительно двигает продукт, а не просто ранжирует клиентов.
- **Batch + realtime:** ночной Airflow-DAG пересчитывает скоры по всей базе; realtime-ветка обрабатывает ключевые события клиента (пополнение, первая покупка, изменение риск-профиля) в режиме «минут».

## Формулировки для резюме

Длинная:

> **Спроектировал и внедрил** систему сегментации клиентов и персонализации инвест-предложений (Next Best Offer) для CRM/маркетинга инвесткомпании. Построил гибридный пайплайн: **HDBSCAN-сегментация** по поведенческим, портфельным и KYC-фичам + **CatBoost/LightGBM-скоринг** вероятности отклика на каждый из инвестпродуктов (акции/облигации, ПИФ/БПИФ, ИИС, ДУ). NBA выбирается как argmax по вектору скоров с наложением бизнес-правил по риск-профилю и комплаенсу. Работа в двух режимах — **ночной batch в Airflow** + **realtime-досчёт** при ключевых событиях клиента. Обеспечил интерпретируемость для маркетинга через **SHAP** и портреты сегментов. Замерил эффект через **A/B-тест** (**uplift**, CTR, conversion), качество моделей — через **ROC-AUC / PR-AUC** и кластерные метрики (silhouette, Davies-Bouldin).

Короткая:

> Гибрид HDBSCAN-сегментации и CatBoost/LightGBM-скоринга для Next Best Offer в ИК; batch (Airflow) + realtime-досчёт при событиях клиента; интерпретация через SHAP, замер uplift'а в A/B. Заменил веерный маркетинг на персонализированные коммуникации.
