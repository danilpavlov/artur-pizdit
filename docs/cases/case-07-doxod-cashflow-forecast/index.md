# Кейс 07 (мини): Прогноз притока/оттока средств (cashflow / AUM) — ИК «ДОХОДЪ»

!!! info "Мини-кейс"
    Временные ряды. Компактная версия: 3 страницы — обзор, флоу, детали.

## Краткое описание

**Домен:** корпоративные финансы и операционное планирование инвесткомпании «ДОХОДЪ».

**Задача:** прогнозировать **ежедневный и еженедельный приток/отток средств клиентов** (пополнения и выводы) — по компании в целом и по ключевым сегментам — чтобы казначейство и операционная команда могли планировать **ликвидность**, а маркетинг — таймировать коммуникации под «окна» активности.

**Итог одной строкой:** гибрид **Prophet / ETS** как baseline + **LightGBM с лаг-фичами и календарём** как основная модель; дневной горизонт до 14 дней, недельный до 8 недель; оценка через MAPE / MAE / бэктест на скользящем окне; результат идёт в дашборд казначейства и в планирование маркетинговых активностей.

## Почему это нетривиально

- **Сильная сезонность:** недельная (будни/выходные), месячная (зарплатные дни), годовая (новогодние периоды, майские).
- **Календарные аномалии:** дивидендные отсечки, дни экспирации, офсеты праздников, объявления ЦБ.
- **Редкие шоки:** рыночные события (всплески волатильности), регуляторные новости — их нельзя «выучить», но нужно либо маркировать, либо устойчиво обрабатывать через робастные лосс-функции.
- **Два горизонта:** дневной для операционки, недельный для планирования — это де-факто **разные модели** с разными наборами фичей.

## Формулировка для резюме

> **Разработал** модели прогноза ежедневного и еженедельного притока/оттока клиентских средств инвесткомпании. Начал с baseline **Prophet / ETS**, основная модель — **LightGBM** с лаг-фичами, календарными признаками (зарплатные дни, праздники, дивидендные отсечки) и индикаторами рыночных событий. Валидация — **rolling-origin бэктест**, метрики **MAPE / MAE**. Прогноз используется казначейством для **планирования ликвидности** и маркетингом для таймирования коммуникаций.
