# Кейс 08: Dynamic pricing для селлера на WB/Ozon (Webbee ML)

## Краткое описание

**Работодатель:** Webbee ML — аутсорс ML-решений с фокусом на e-commerce.

**Клиент:** селлер, продающий на маркетплейсах **Wildberries и Ozon** — сотни–тысячи SKU разных категорий (конкретная категория и название клиента обезличены).

**Задача:** построить систему **автоматической рекомендации цен по SKU**, которая максимизирует **маржу**, не давая при этом просесть объёму продаж в штуках больше, чем на заданный процент относительно baseline. Учитывает акции и промо-календари маркетплейсов, out-of-stock периоды, и даёт категорийному менеджеру короткое объяснение по каждой рекомендации.

**Итог одной строкой:** трёхчастная архитектура **forecast спроса (LightGBM global model) → оценка ценовой эластичности по SKU/кластеру → оптимизация цены с ограничением на объём**; NLP-подсистема для кластеризации товаров по описаниям (cold-start и pooled-эластичность) и product matching с ценами конкурентов; ежедневный batch в Airflow, публикация цен в WB/Ozon через API селлера; эффект измерен через **бэктест + A/B по SKU**.
