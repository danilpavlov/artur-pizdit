# Достижения

## Бизнес-результаты

- **Ручной скрининг откликов заменён автоматическим ранжированием.** HR больше не просматривает все отклики подряд — работает сверху вниз по отсортированному списку.
- **Сильные кандидаты не теряются в хвосте.** До внедрения HR физически не доходил до конца длинной ленты откликов; теперь релевантные кандидаты гарантированно попадают в топ.
- **Консистентность оценки между HR-менеджерами.** Базовый порядок выдачи один и тот же, субъективность HR уходит в верхний уровень принятия решения, а не в «кого я вообще увидел».
- **Объяснимость.** Каждое резюме в выдаче сопровождается коротким объяснением от LLM-reranker'а — HR видит, **почему** кандидат на своём месте, а не просто число.

## Технические результаты

- **Гибридный ранжировщик:** BM25 (`rank_bm25`) + dense (Qdrant с дообученным эмбеддером) + LLM-reranker (Qwen 72B через SGLang).
- **Метрики качества ранжирования:** NDCG@k, MAP@k на hold-out из исторических наймов и размеченных HR пар (точные числа — TODO, заполнить актуальными значениями).
- **Дообученный эмбеддер** под HR-домен: позитивы из истории наймов + синтетика от self-hosted LLM + hard-negative mining.
- **Latency-оптимизация:** жёсткое ограничение top-K (30–50 кандидатов) перед LLM-reranker'ом + батчинг запросов в SGLang — иначе 72B физически не уложилась бы в SLA.

## Формулировки для резюме

Длинная:

> **Спроектировал и внедрил** систему автоматического ранжирования резюме с hh.ru для HR-команды инвестиционной компании. Архитектура — **двухступенчатый ранжировщик**: гибридная кандидатогенерация (BM25 через `rank_bm25` + dense-поиск в **Qdrant** с дообученным русскоязычным sentence-transformer'ом) → LLM-reranker на **Qwen 72B через SGLang**, выдающий score и объяснение по каждой паре «вакансия ↔ резюме». Дообучил эмбеддер на исторических наймах + синтетических парах от self-hosted LLM с hard-negative mining. Оценивал качество через **NDCG@k / MAP@k**. Заменил ручной скрининг откликов HR-менеджерами и поднял релевантных кандидатов в топ выдачи.

Короткая:

> Гибридный ранжировщик резюме (BM25 + Qdrant + Qwen 72B reranker на SGLang) с дообученным под HR-домен эмбеддером; заменил ручной скрининг откликов с hh.ru; оценка по NDCG@k / MAP@k.
