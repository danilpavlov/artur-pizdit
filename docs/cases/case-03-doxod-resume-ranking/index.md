# Кейс 03: Ранжирование резюме с hh.ru по релевантности вакансии (ИК «ДОХОДЪ»)

## Краткое описание

**Домен:** внутренний HR инвестиционной компании «ДОХОДЪ».

**Задача:** автоматизировать скрининг откликов на вакансии с hh.ru — отранжировать сотни входящих резюме по релевантности структурированному профилю вакансии, чтобы HR видел топ-N кандидатов сверху и не тратил часы на ручной просмотр.

**Итог одной строкой:** гибридный ранжировщик **BM25 + дообученные эмбеддинги в Qdrant + Qwen 72B через SGLang как reranker**; self-hosted в контуре компании; заменил ручной скрининг откликов и вывел релевантных кандидатов в топ выдачи HR-интерфейса.
